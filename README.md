## 👋 About Me

**电子科技大学 (UESTC)** 在读硕士研究生，专注于安全研究与 AI Safety。

我的研究兴趣横跨底层系统安全与人工智能安全两大领域——从 Android 内核逆向到大模型多智能体安全，探索攻防对抗的本质。
